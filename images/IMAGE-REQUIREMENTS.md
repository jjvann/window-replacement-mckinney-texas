## üìê Image Specifications

### Recommended Dimensions:
- **Hero Images**: 1920√ó1080px (16:9 aspect ratio)
- **Card Images**: 800√ó600px (4:3 aspect ratio)
- **Content Images**: 1200√ó800px (3:2 aspect ratio)
- **Project Images**: 1200√ó800px (3:2 aspect ratio)

### File Formats:
- **Primary**: JPG (for photos)
- **Alternative**: WebP (for better compression)
- **Transparency**: PNG (if needed for logos/graphics)

### File Size Guidelines:
- **Hero Images**: < 500KB
- **Card Images**: < 200KB
- **Content Images**: < 300KB

### Naming Convention:
- Use lowercase with hyphens
- Be descriptive: `hero-image.jpg`, `installation-1.jpg`
- Include numbers for multiple: `example-1.jpg`, `example-2.jpg`

---

## üí° Tips

1. **Reuse Images**: Service card images can be used on homepage
2. **Start Small**: Begin with Phase 1, then expand
3. **Quality Over Quantity**: Better to have fewer high-quality images
4. **Consistent Style**: Keep lighting, composition, and style consistent
5. **Optimize**: Compress images before uploading for faster page loads
