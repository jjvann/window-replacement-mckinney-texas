# Images Directory Structure

This directory contains image folders organized by service type and location.

## Services Images
Place service-specific images in the corresponding service folders:
- `services/double-hung-windows/` - Images for double-hung window pages
- `services/single-hung-windows/` - Images for single-hung window pages
- `services/casement-windows/` - Images for casement window pages
- `services/awning-windows/` - Images for awning window pages
- `services/sliding-windows/` - Images for sliding window pages
- `services/picture-windows/` - Images for picture window pages
- `services/bay-bow-windows/` - Images for bay & bow window pages
- `services/special-shape-windows/` - Images for special shape window pages

## Location Images
Place location-specific images in the corresponding location folders:
- `locations/fort-worth/` - Images for Fort Worth location pages
- `locations/arlington/` - Images for Arlington location pages
- `locations/grand-prairie/` - Images for Grand Prairie location pages
- `locations/irving/` - Images for Irving location pages
- `locations/euless/` - Images for Euless location pages
- `locations/bedford/` - Images for Bedford location pages
- `locations/hurst/` - Images for Hurst location pages
- `locations/grapevine/` - Images for Grapevine location pages
- `locations/colleyville/` - Images for Colleyville location pages
- `locations/southlake/` - Images for Southlake location pages
- `locations/keller/` - Images for Keller location pages
- `locations/north-richland-hills/` - Images for North Richland Hills location pages
- `locations/haltom-city/` - Images for Haltom City location pages
- `locations/richland-hills/` - Images for Richland Hills location pages
- `locations/watauga/` - Images for Watauga location pages

## Recommended Image Types
- Hero images (1920x1080 or similar)
- Product/window examples
- Before/after photos
- Installation photos

## File Naming Convention
Use descriptive, lowercase filenames with hyphens:
- `hero-image.jpg`
- `window-example-1.jpg`
- `installation-photo.jpg`
